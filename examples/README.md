# Examples

This directory contains MapLibre GL 3D Tiles examples.

## Basic Example

Vanilla TypeScript example that adds `ThreeDTilesControl` to a MapLibre map and loads the default public tileset.

```bash
npm run dev
```

Open http://localhost:5173/examples/basic/.

## React Example

React example that uses `ThreeDTilesControlReact` and `useThreeDTilesState`.

```bash
npm run dev
```

Open http://localhost:5173/examples/react/.

## Build

```bash
npm run build:examples
```

The built examples are written to `dist-examples`.
